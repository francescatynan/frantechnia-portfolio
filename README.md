# Frantechnia Portfolio - by Francesca Tynan - README

Hi, I'm Francesca Tynan, software engineer by trade since 1st May 2025, passionate about tech, travel, food and life as a software engineer with an interest in cyber security. I currently work for [ThinkTMB](https://thinktmb.com/).

Deploying this January 2026 marks a milestone. This is my first portfolio redesign since becoming a professional software engineer, my first design was a much simpler iteration written before my tech bootcamp with [Code First Girls](https://codefirstgirls.com/) and this is an exciting opportunity to update my portfolio to reflect everything I've learnt in 8 months of professional employment and reflecting around 2 years and 3 months since I began an entry-level course towards this career path, 1 year and 8 months since I began the tech bootcamp that helped me gain the skills to navigate this career switch.

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app). I decided to utilise the default recommended config to build upon, because this is my first Next.js app and npm as the Javascript package manager choice helped keep things simple, whilst I focused my interests on a clean build and how to be more creative with styling, dark/light mode choice and creating a solid platform to showcase future projects.

## Getting Started

First, run the development server:

```bash
npm run dev

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
```

See package.json "scripts" for more.
